# Installation
#### Git repository
```
$ git clone https://github.com/Empressive/intranet-tasks.git
$ cd intranet-tasks
```
#### Requirements
Pip
Python 3.5
Django 2.0

#### Virtual enironment
```
$ python3.5 -m venv .env
$ source ./env/bin/activate
```

#### Pip packages
```
(.env)$ pip install -r requirements.txt
```

#### Launching
```
(.env)$ python3 manage.py runserver
```
